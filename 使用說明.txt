================================================================
  pdftomd — PDF 轉 Markdown 工具  使用說明
================================================================

【第一次使用：安裝】
1. 解壓縮此 zip 到任意資料夾（例如 D:\pdftomd）
2. 在 install.ps1 上按右鍵 → 以 PowerShell 執行
3. 若出現「是否要執行」詢問，選「是」或「開啟」
4. 等待安裝完成（需要網路，約 15~30 分鐘）
   - 安裝 Python 套件：5~15 分鐘
   - 下載 AI 模型：10~20 分鐘（約 1~2 GB）
5. 看到「安裝完成！」即可開始使用

  ⚠️ 安裝只需做一次，之後直接雙擊 .bat 即可。

----------------------------------------------------------------

【使用方式：講義類 PDF（教科書、掃描、圖表多）】

  步驟：
  1. 把 PDF 放入  note\input\  資料夾
  2. 雙擊  run_mineru.bat
  3. 等待轉換完成
  4. 結果在  note\output\  資料夾
     - Markdown 檔：note\output\{檔名}.md
     - 圖片：note\output\images\

----------------------------------------------------------------

【使用方式：技術文件類 PDF（程式書、說明書、純文字）】

  步驟：
  1. 把 PDF 放入  dev\input\  資料夾
  2. 雙擊  run_docling.bat
  3. 等待轉換完成
  4. 結果在  dev\output\  資料夾
     - Markdown 檔：dev\output\{檔名}.md
     - 圖片：dev\output\images\

----------------------------------------------------------------

【常見問題】

Q：出現「無法載入指令碼」的 PowerShell 錯誤？
A：在 install.ps1 上按右鍵 → 內容 → 解除封鎖，再重新執行。

Q：轉換速度很慢？
A：首次轉換需載入 AI 模型，稍慢屬正常。後續轉換會快很多。

Q：可以一次放多個 PDF 嗎？
A：可以，直接放多個 PDF 進 input 資料夾，會依序全部轉換。
